# Roblox Follow Bot

A Python script that sends Roblox follows to a target user using cookies and proxies.

## How to run

1. Add one or more Roblox `.ROBLOSECURITY` cookies to `cookies.txt` (one per line).
2. Add proxies to `proxies.txt` (one per line, in `host:port` format).
3. Run the workflow — it will prompt for a Roblox user ID in the console.

```
python main.py
```

## Files

- `main.py` — main script
- `cookies.txt` — Roblox security cookies (one per line)
- `proxies.txt` — proxies to route requests through (one per line)

## Dependencies

- `requests` (installed via pip)
